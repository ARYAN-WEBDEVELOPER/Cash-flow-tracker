🤖 AI Prompts Used

Below are the prompts used to build this project:

Project Setup
"Create a Cash Flow Tracker web app using HTML, CSS, and JavaScript with salary input, expense tracking, and balance calculation."

Expense Management Logic
"Write JavaScript logic to add, delete, and display expenses dynamically using DOM manipulation."

LocalStorage Integration
"Store salary and expense data in localStorage and retrieve it on page reload."

Real-Time Calculation
"Calculate total expenses and remaining balance dynamically when user adds or deletes expenses."

Chart Visualization
"Use Chart.js to create a pie chart showing expenses vs remaining balance."

PDF Export
"Generate a downloadable expense report PDF using jsPDF with all expenses and final balance."

Currency Converter
"Implement currency conversion (INR to USD/EUR) using a free API and update values in real-time."

Budget Alert System
"Show a warning when remaining balance goes below 10% of salary."

Toast Notification System
"Create a custom toast popup system for success, error, and warning messages."

UI/UX Improvements
"Design a modern, responsive UI for the dashboard with better styling and layout."

Bug Fixing & Debugging
"Fix issues related to function scope, duplicate functions, currency display, and PDF encoding errors."
